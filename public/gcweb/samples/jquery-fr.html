Chargement de jQuery

SGDC vous offre la possibilité de charger jQuery. Par défaut, jQuery est chargé localement à partir de la version WET. Cependant, vous avez également la possibilité de charger jQuery à partir de Google CDN.

Mise en oeuvre

Vous n'avez rien à faire si vous souhaitez utiliser l'option par défaut de chargement de jQuery à partir de la version WET.

Le chargement de jQuery à partir de Google CDN se fait via le paramètre jQuery dans la section refFooter de votre page.

document.write(wet.builder.refFooter({
...
    "jqueryEnv": external
...
});

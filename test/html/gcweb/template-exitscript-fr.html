Quitter un site sécurisé

  • Ce lien https://www.google.ca a un nom de domaine (www.google.ca) différent du vôtre. Vous serez redirigé sur la page exiturl-fr.html car le paramètre exitURL contient la valeur exiturl-fr.html.
  • Ce lien https://www.google.ca s'ouvrira dans une autre fenêtre. Grâce au paramètre targetWarning, vous pouvez définir un message qui avertira les utilisateurs que le lien s'ouvrira dans une autre fenêtre. Ce message ajoutera le message d'affichage existant.
  • Ce lien breadcrumbs-fr.html a le même nom de domaine que le vôtre. Vous ne serez pas redirigé sur la page exiturl-fr.html.
  • Ce lien https://google.com a son nom de domaine (google.com) dans la liste d'exception exitDomains. Vous ne serez pas redirigé sur la page exiturl-fr.html.

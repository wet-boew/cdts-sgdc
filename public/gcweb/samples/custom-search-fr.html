Recherche personnalisée

Lorsque vous utilisez le modèle d'application, vous avez la possibilité d'utiliser un moteur de recherche personnalisé pour votre site.

Mise en oeuvre

La personnalisation de la recherche se fait via le paramètre customSearch dans la section appTop de votre page.

Il existe quelques options de configuration:

  • action: l'URL de la recherche vers laquelle vous souhaitez pointer
  • method: get ou post
  • placeholder: Facultatif - texte d'espace réservé affiché dans la barre de recherche
  • id: Facultatif - vous pouvez fournir un identifiant facultatif pour votre recherche
  • name: Facultatif - vous pouvez personnaliser l'attribut name pour votre recherche
  • hiddenInput: Facultatif - vous pouvez fournir des entrées de données via les paramètres name et value
defTop.outerHTML = wet.builder.appTop({
...
    "customSearch": [{
        "action" : "https://www.canada.ca/en/sr/srb.html",
        "placeholder" : "Canada.ca",
        "id" : "wb-srch-q",
        "name" : "q",
        "method" : "get",
        "hiddenInput" : [{
            "name" : "cdn",
            "value" : "canada"
        }]
    }]
...
});

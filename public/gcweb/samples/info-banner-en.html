Information Banner

This sample page will demontrate how to implement an information banner on top of the page.

Implementation

The banner is added in the appTop section of the page with the following configuration options:

  • "showBanner": the url of the link.
  • "mainHTML": the text you would like displayed in your custom banner.
  • "link":"text": the text of the link that is displayed.
  • "link":"href": the url of the link.
  • "button":"text": the text of the button that is displayed.
  • "button":"href": the url of the button.
defTop.outerHTML = wet.builder.appTop({
...
    "infoBanner": {
        "mainHTML": "Beta version: We've made some changes to improve your experience with My Service Canada Account.",
        "link": {
            "text": "Learn more about the beta version",
            "href": "#"
        },
        "button": {
            "text": "Try beta version",
            "href": "#"
        }
    }
...
});

Lien de contact

La variable de lien de contact remplacera le lien par défaut « Contactez avec nous » situé dans le pied de page de votre page. Si aucune variable n'est spécifiée, l'URL par défaut sera utilisée. Vous pouvez également modifier le texte du lien. Vous avez également la possibilité d'ajouter plusieurs liens.

Mise en oeuvre

Le lien de contact est modifié dans la section footer de la page. Vous avez deux options de configuration:

  • text: le texte d'affichage du lien
  • href: l'url du lien
defFooter.outerHTML = wet.builder.footer({
...
    "contactLinks": [{
        "href": "#",
        "text": "Lien de contact 1"
    }, {
        "href": "#",
        "text": "Lien de contact 2"
    }, {
        "href": "#",
        "text": "Lien de contact 3"
    }]
...
});

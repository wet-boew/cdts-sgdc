Bouton de connexion

Cet exemple de page montre comment implémenter un bouton de connexion.

Mise en œuvre

Le bouton de connexion est ajouté dans la section appTop de la page.

Vous pouvez fournir l'URL vers laquelle vous souhaitez rediriger l'utilisateur via le paramètre href

defTop.outerHTML = wet.builder.appTop({
...
    "signIn": [{"href": "#"}]
...
});

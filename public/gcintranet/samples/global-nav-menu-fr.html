Menu de navigation globale

Un menu de navigation global est un pied de page global qui peut être affiché à la place du pied de page par défaut.

Mise en oeuvre

Le menu de navigation global est ajouté dans la section appFooter de la page.

Définissez simplement la variable globalNav sur true.

defFooter.outerHTML = wet.builder.appFooter({
...
    "globalNav": true
...
});

Quitter un site sécurisé

  • Ce lien http://www.alloprof.qc.ca/ affichera un avertissement puisque son nom de domaine (www.alloprof.qc.ca) est différent du vôtre. Et il vous redigera sur la page refFooter_exiturl-fr.shtml car le paramètre refFooter_exitURL contient la valeur exiturl-fr.shtml.
  • Ce lien apptop_breadcrumbs-fr.shtml n'affichera pas un avertissement car son nom de domaine est le même que le vôtre.
  • Ce lien https://developer.mozilla.org/fr/docs/Apprendre/a11y n'affichera pas un avertissement car son nom de domaine (developer.mozilla.org) est dans la liste d'exception exitDomains.

Leaving a secure site

  • This link https://www.google.ca will display a warning because his hostname (google.ca) is different from yours. You'll be redirect to the exiturl-en.html page because the exitURL parameter contains the value exiturl-en.html.
  • This link https://www.google.ca will open in another window. Through the targetWarning parameter, you can set a message that will warn users that the link will open in another window. This message will append the existing display message.
  • This link breadcrumbs-en.html won't display a warning because it has the same hostname as yours.
  • This link https://google.com won't display a warning because his hostname (google.com) is in the exception list exitDomains.

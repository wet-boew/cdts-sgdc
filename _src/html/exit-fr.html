Description of page

The link http://csszengarden.com/219/ will display a warning because his hostname (csszengarden.com) is different from yours. You'll be redirect to the exiturl-en.html page because the exitURL parameter contains the value exiturl-en.html.

The link ./breadcrumbs-en.html won't display a warning because it has the same hostname as yours.

The link https://developer.mozilla.org/en-US/docs/Web/Accessibility/Understanding_WCAG won't display a warning because his hostname (developer.mozilla.org) is in the exception list exitDomains.

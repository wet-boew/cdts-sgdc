Note

Voici la foire aux questions pour la SGDC. N'hésitez pas à rajouter des questions!

  • 1. Quand sortira la prochaine version de la SGDC?

    La prochaine version est toujours annoncée sur toutes les pages de la SGDC, sous le menu. Nous suivons normalement la feuille de route de la BOEW, à moins de devoir corriger des problèmes ou de présenter de nouveaux gabarits. Ils sont cependant toujours annoncés à l'avance!

  • 2. Quel gabarit devrais-je utiliser?

    Nous avons placé un arbre de décision pour vous aider à faire votre choix sur le gabarit qui vous convient sur la page d'accueil.

  • 3. Est-ce que je peux retirer le bouton « Signaler un problème ou une erreur sur cette page » ?

    Il y a trois scénarios possibles pour ce bouton :

    1. Il peut être omis pour les applications Web
    2. Il peut contenir un lien personnalisé (redirige les utilisateurs vers votre propre page contactez-nous/aide).
    3. Il peut contenir le lien par défaut (redirige vers une page générique de Canada.ca qui pourrait rediriger ou non le message à la bonne destination). Le lien par défaut est surtout utilisé pour des commentaires généraux à propos de Canada.ca et n'a PAS été conçu pour le soutien d'applications précises.

    Voyez comment vous pouvez personnaliser ou désactiver le bouton. [Lien]

  • 4. Sous quelles conditions puis-je utiliser le scénario d'opérations (mode transactionnel)?

    Sur les pages où l’utilisateur effectue une opération — et où lui permettre de quitter la page active interromprait le cheminement souhaité et donnerait lieu à des erreurs, à la perte de données ou à la fin accidentelle de la session — les éléments de l’en-tête et du pied de page peuvent être supprimés.

  • 5. Puis-je obtenir mon propre thème personnalisé?

    Certainement! Si votre actif Web est interne (intranet) ou si vous désirez obtenir un champ de recherche précis et/ou un pied de page pour la version application de la SGDC, veuillez nous contacter et il nous fera plaisir de vous aider!

  • 6. Est-ce que la SGDC offre le code de Google Analytics?

    Nous avons supprimé le code de Google Analytics parce que Canada.ca est en train de mettre en place son propre outil Adobe Analytics. Chaque application Web est libre de mettre en place leur propre outil de statistiques. Ce ne sera pas imposé par la SGDC pour le moment.

  • 7. Sur quoi puis-je travailler maintenant que je ne gère plus la couche de pésentation? J'ai tellement de temps pour moi maintenant que j'ai mis en oeuvre la SGDC!

    Nous vous suggérons de consulter Twitter et de nous suivre @SGDC-CDTS. Ensuite, le site XKCD devrait vous tenir occupé pour quelques heures! Y a-t-il vraiment quelqu'un qui lit ces FAQ?

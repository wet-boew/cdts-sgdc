Header Menu

This sample page demonstrates how a header menu can be displayed on your page.

Implementation

The menu is added in the appTop section of the page with the following configuration options:

  • "text": (MANDATORY) - the text inside the menu
  • "link":"text": (MANDATORY) - the text of the link that is displayed
  • "link":"href": (MANDATORY) - the url of the link
  • "link":"newWindow": (OPTIONAL) - open the link in a new window
  • "logoutLink":"text": (OPTIONAL) - the text of the logout link that is displayed.
  • "logoutLink":"href": (OPTIONAL) - the url of the logout link
defTop.outerHTML = wet.builder.appTop({
...
    "headerMenu": {
        "text": "Account",
        "links": [{
            "href": "custom-menu-fr.html",
            "text": "Link 1"
        },
        {
            "href": "custom-menu-fr.html",
            "text": "Link 2",
            "newWindow": true
        }],
        "logoutLink": {
            "href": "#",
            "text":"Logout Link",
        }
    }
...
});

Bannière d'information

Cet exemple de page montre comment implémenter une bannière d'information en haut de la page.

Mise en œuvre

La bannière est ajoutée dans la section appTop de la page avec les options de configuration suivantes :

  • "mainHTML": the text you would like displayed in your custom banner.
  • "link":"text": le texte du lien affiché.
  • "link":"href": l'url du lien.
  • "link":"newWindow": ouvrir le lien dans une nouvelle fenêtre.
  • "button":"text": le texte du bouton affiché.
  • "button":"href": l'url du bouton.
  • "button":"newWindow": ouvrir le lien dans une nouvelle fenêtre.
defTop.outerHTML = wet.builder.appTop({
...
    "infoBanner": {
        "mainHTML": "Version bêta : Nous avons apporté quelques changements pour améliorer votre expérience avec Mon dossier Service Canada.",
        "link": [{
            "text": "En savoir plus sur la version bêta",
            "href": "#",
            "newWindow": true
        }],
        "button": {
            "text": "Essayez la version bêta",
            "href": "#",
            "newWindow": true
        }
    }
...
});

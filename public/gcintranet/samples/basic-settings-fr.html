Paramètres de base

Cet exemple de page fournit des informations pour configurer les éléments de base de votre page.

Date modifiée

La date de modification est affichée en bas de la page, au-dessus du pied de page.

Ceci est configuré dans la section preFooter de votre page.

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "dateModified": "2019-07-31"
...
});

Identifiant d'écran

L'identifiant d'écran s'affiche en bas de la page. Cet élément sert un identifiant unique pour votre page et peut être utilisé pour faciliter la communication entre les utilisateurs et le centre de services / l'équipe de support afin d'identifier l'emplacement exact d'un utilisateur dans votre application.

Ceci est configuré dans la section preFooter de votre page.

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "screenIdentifier": "0123456789"
...
});

Identifiant de version

L'identifiant de version est affiché en bas de la page, au-dessus du pied de page.

Ceci est configuré dans la section preFooter de votre page.

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "versionIdentifier": "7890123456"
...
});

Description de la page

Ce lien http://www.alloprof.qc.ca/ affichera un avertissement puisque son nom de domaine (www.alloprof.qc.ca) est différent du vôtre. Et il vous redigera sur la page exiturl-fr.html car le paramètre exitURL contient la valeur exiturl-fr.html.

Ce lien ./breadcrumbs-fr.html n'affichera pas un avertissement car son nom de domaine est le même que le vôtre.

Ce lien https://developer.mozilla.org/fr/docs/Apprendre/a11y n'affichera pas un avertissement car son nom de domaine (developer.mozilla.org) est dans la liste d'exception exitDomains.

Page transactionnelle

Une page transactionnelle doit être utilisée dans des scénarios spécifiques. Le Conseil du Trésor définit cela comme suit:

Sur les pages où l’utilisateur effectue une opération – et où lui permettre de quitter la page active interromprait le cheminement souhaité et donnerait lieu à des erreurs, à la perte de données ou à la fin accidentelle de la session.

Fondamentalement, vous évitez d'avoir des liens externes pour empêcher les utilisateurs de perdre des données s'ils devaient s'éloigner.

Mise en œuvre

Quelques modifications sont nécessaires pour rendre votre page vraiment transactionnelle. Tous ces changements sont obligatoires, sauf indication contraire.

Dans la section top:

  • Masquer la barre de recherche
    • defTop.outerHTML = wet.builder.top({
...
    "search": false
...
});
  • Masquer le menu du site
    • defTop.outerHTML = wet.builder.top({
...
    "siteMenu": false
...
});
  • Le fil d'Ariane doit être remplacé par un lien pour quitter la transaction qui dirige les utilisateurs vers un point sûr où ils peuvent abandonner la transaction en cours et la redémarrer s'ils le souhaitent.
    • defTop.outerHTML = wet.builder.top({
...
    "breadcrumbs": [{
        "title": "Exit secure transaction",
        "href": "https://www.canada.ca/en/index.html"
    }]
...
});
  • Masquer le contenu inséré dans l'espace de pré-contenu de votre page
    • defTop.outerHTML = wet.builder.top({
...
    "showPreContent": false
...
});

Dans la section preFooter:

  • Masquer le contenu inséré dans l'espace de contenu de publication de votre page
    • defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showPostContent": false
...
});
  • Masquer le bouton « Signaler un problème ou une erreur sur cette page »
    • defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showFeedback": false
...
});
  • Masquer le bouton « Partager cette page »
    • defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showShare": false
...
});

Dans la section footer:

  • Masquer le contenu à l'intérieur du pied de page
    •     defFooter.outerHTML = wet.builder.footer({
    ...
        "showFooter": false
    ...
    });
  • Facultatif: remplacez le lien « Coordonnées », le lien « Avis » et le lien « Confidentialité » situé au bas de la page. Un paramètre optionnel newWindow peut être défini sur true si vous souhaitez que le lien s'ouvre dans une nouvelle fenêtre.
  • Lien Coordonnées
    • defFooter.outerHTML = wet.builder.footer({
...
    "contactLink": [{
        "href": "https://www.canada.ca/en/contact.html",
        "newWindow": true
    }]
...
});
  • Lien Avis
    • defFooter.outerHTML = wet.builder.footer({
...
    "termsLink": [{
        "href": "https://www.canada.ca/en/transparency/terms.html",
        "newWindow": true
    }]
...
});
  • Lien Confidentialité
    • defFooter.outerHTML = wet.builder.footer({
...
    "privacyLink": [{
        "href": "https://www.canada.ca/en/transparency/privacy.html",
        "newWindow": true
    }]
...
});

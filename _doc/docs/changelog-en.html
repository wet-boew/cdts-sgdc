All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project adheres to Semantic Versioning. Some small modifications made so that we can easily copy/paste to GCPedia.

Announcements

  • If you previously implemented the application templates you will NOT be able to upgrade without updating your code. A mandatory update of the backup static files will also be required. Please see GCPedia for updated code and static files. * RL
  • In both themes the /rn/ version is now a copy of the /app/ version. This reduces development time and ensures greater consistency between both versions. * RL
  • All /rn/ static files (GCintranet and GCweb) must be updated before WET v4.1.0. * RL
  • All GCWeb implementations on ssl-templates.services.gc.ca will stop working shortly, make sure you are using the Akamai URLs. * RL
  • Akamai servers for CDTS are now serving up static HTML files, please make sure to grab files from Akamai and not ssl-templates.services.gc.ca. * RL
  • In v4.0.24 – v4.0.25 the HTML for the top menu in the application templates has changed (GCIntranet only), since we’re not aware of anyone except ESDC EI ORT web asset using the application templates on the GCIntranet theme we’ve updated this automatically. If you were impacted don’t forget that you need to let the ESDC WS CoE team know you are using it so we can warn you ahead of time of any changes to the application templates. * RL

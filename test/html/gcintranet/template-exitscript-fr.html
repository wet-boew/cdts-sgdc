Quitter un site sécurisé

  • Ce lien https://www.google.ca affichera un avertissement puisque son nom de domaine (google.ca) est différent du vôtre. Et il vous redigera sur la page exiturl-fr.html car le paramètre exitURL contient la valeur exiturl-fr.html.
  • Ce lien https://www.google.ca s'ouvrira dans une autre fenêtre. Grâce au paramètre targetWarning, vous pouvez définir un message qui avertira les utilisateurs que le lien s'ouvrira dans une autre fenêtre. Ce message ajoutera le message d'affichage existant.
  • Ce lien breadcrumbs-fr.html n'affichera pas un avertissement car son nom de domaine est le même que le vôtre.
  • Ce lien https://google.com n'affichera pas un avertissement car son nom de domaine (google.com) est dans la liste d'exception exitDomains.
  • Ce lien https://www.canada.ca/ a l'attribut cdts-exitscript-disabled l'exemptant de exitscript.

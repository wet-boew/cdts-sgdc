Adobe Analytics

Cet exemple de page montre comment incorporer Adobe Analytics dans votre page Web.

Mise en œuvre

Adobe Analytics est appliqué via le paramètre webAnalytics. Vous aurez une modification à apporter dans la section refTop de votre page en fonction de l'environnement.

Vous devrez également définir webAnalytics sur true dans la section refFooter de votre page.

document.write "refTop" élément

Utiliser Adobe Analytics version 1 sur l'environnement staging.

document.write(wet.builder.refTop({
	"webAnalytics" : [{
		"environment" : "staging",
		"version" : 1
	}]
}));

Utiliser Adobe Analytics version 2 sur l'environnement production.

document.write(wet.builder.refTop({
	"webAnalytics" : [{
		"environment" : "production",
		"version" : 2
	}]
}));

Utiliser Adobe Analytics version 3 sur l'environnement staging.

document.write(wet.builder.refTop({
	"webAnalytics" : [{
		"custom" : "launch-EN11c0261481f74c56b7656937bbd995e9-staging.min.js"
	}]
}));

Utiliser Adobe Analytics version 3 sur l'environnement production.

document.write(wet.builder.refTop({
	"webAnalytics" : [{
		"custom" : "launch-EN0cf6c2810a2b48f8a4c36502a1b09541.min.js"
	}]
}));

document.write "refFooter" élément

document.write(wet.builder.refFooter({
    "webAnalytics" : true
}));

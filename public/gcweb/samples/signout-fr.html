Bouton Fermer la session

Cet exemple de page explique comment implémenter un bouton Fermer la session.

Mise en oeuvre

Le bouton Fermer la session est ajouté dans la section appTop de la page.

Vous pouvez fournir l'URL vers laquelle vous souhaitez rediriger l'utilisateur via le paramètre href

defTop.outerHTML = wet.builder.appTop({
...
    "signOut": [{"href": "#"}]
...
});

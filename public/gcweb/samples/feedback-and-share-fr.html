Commentaires et partage

Cette page montrera comment activer / désactiver le lien de rétroaction, ainsi que comment utiliser la fonction pour partager la page actuelle avec certaines applications de médias sociaux prédéfinies.

Rétroaction

Spécifiez le bouton « Signaler un problème ou une erreur sur cette page » de votre page. Par défaut, ce bouton est affiché avec l'URL par défaut de Canada.ca. Vous pouvez personnaliser l'URL ou simplement masquer le bouton.

Mise en oeuvre

Ce lien peut être personnalisé dans la section preFooter de votre page.

Cacher le bouton

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showFeedback": false
...
});

Personnalisez l'URL

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showFeedback": "Custom URL here"
...
});

Partager

Le bouton « Partager cette page » est un moyen pour les utilisateurs de partager l'URL du site via les médias sociaux. Par défaut, ce bouton s'affiche avec la sélection par défaut des sites de réseaux sociaux. Vous pouvez personnaliser cette sélection ou simplement masquer le bouton.

Mise en oeuvre

Ce bouton peut être personnalisé dans la section preFooter de votre page.

Cacher le bouton

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showShare": false
...
});

Choisissez les sites de médias sociaux que vous souhaitez afficher comme options pour l'utilisateur.

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showShare": ["email", "facebook", "linkedin", "twitter"]
...
});

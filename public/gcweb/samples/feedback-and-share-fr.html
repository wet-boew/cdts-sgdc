Commentaires et partage

Cette page montrera comment activer l'outil de rétroaction sur la page, ainsi que comment utiliser la fonction pour partager la page actuelle avec certaines applications de médias sociaux prédéfinies.

Rétroaction

À partir de la version 4.2.0, SGDC remplace le bouton « Signaler un problème » par le nouvel outil de rétroaction de page fourni par WET. Par défaut, l'outil de commentaires sur la page ne sera pas affiché. Il faudra l'allumer. En outre, l'outil de rétroaction sur la page peut être personnalisé pour ajouter un lien de contact de votre choix.

Pour plus d'informations, veuillez consulter la documentation WET ici.

Mise en oeuvre

Cet outil de commentaires sur la page peut être ajouté et personnalisé dans la section preFooter de votre page.

Afficher l'outil de rétroaction sur la page

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showFeedback": true
...
});

Personnalisez l'outil pour ajouter un lien de contact

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showFeedback": {
        "text": "Contact Name",
        "url": "Contact URL"
    }
...
});

Mettre à jour les valeurs des champs de saisie masqués : Thème et Section

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showFeedback": {
        "Theme": "Text defining the theme of your page",
        "Section": "Text defining the section where your page resides"
    }
...
	});

Note: Vous pouvez définir les métadonnées suivantes : <meta name="dcterms.creator" content="[Department name]">

Note: Les valeurs pour le thème, la section et le nom du département doivent être les MÊMES pour l'anglais et le français.

Partager

Le bouton « Partager cette page » est un moyen pour les utilisateurs de partager l'URL du site via les médias sociaux. Par défaut, ce bouton s'affiche avec la sélection par défaut des sites de réseaux sociaux. Vous pouvez personnaliser cette sélection ou simplement masquer le bouton.

Mise en oeuvre

Ce bouton peut être personnalisé dans la section preFooter de votre page.

Cacher le bouton

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showShare": false
...
});

Choisissez les sites de médias sociaux que vous souhaitez afficher comme options pour l'utilisateur.

defPreFooter.outerHTML = wet.builder.preFooter({
...
    "showShare": ["email", "facebook", "linkedin", "twitter"]
...
});

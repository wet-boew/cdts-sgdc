Modal GCTools

Cet exemple de page montre comment afficher un modal contenant les liens GCTools

Mise en œuvre

Par défaut, quatre liens GCTools s'affichent dans l'en-tête. Vous pouvez remplacer tout cela par un seul lien vers une boîte de dialogue modale qui contiendra les liens GCTools à la place.

Cela peut être fait via le paramètre GCToolsModal dans la section top de votre page.

document.write(wet.builder.top({
...
    "GCToolsModal": true
...
});

Footer

Depuis la v4.0.47, SGDC implémente la dernière version du pied de page du site fournie par WET.

Pour plus d'informations, veuillez consulter la documentation du pied de page WET ici.

Bande contextuelle

La bande contextuelle est masqué par défaut. Rendre visible la bande contextuelle se fait via le paramètre contextualFooter.

Ceci est configuré dans la section footer de votre page.

  • "title": le titre de la bande contextuelle.
  • "text": le texte du lien affiché.
  • "href": l'url du lien.
defFooter.outerHTML = wet.builder.footer({
...
    "contextualFooter":{
        "title": "Title",
        "links": [{
            "text": "Link 1",
            "href":"#"
        },
        {
            "text": "Link 2",
            "href":"#"
        },
        {
            "text": "Link 3",
            "href":"#"
        },
        {
            "text": "Link 4",
            "href":"#"
        }]
    }
...
});

La bande principale

La bande principale est visible par défaut. Vous avez la possibilité de masquer la bande principale via le paramètre hideFooterMain.

Ceci est configuré dans la section footer de votre page.

> 
defFooter.outerHTML = wet.builder.footer({
...
    "hideFooterMain": true
...
});

La bande sous le pied de page

La bande sous le pied de page est visible par défaut. Vous avez la possibilité de masquer les trois premiers liens via le paramètre hideFooterCorporate.

Vous avez également la possibilité d'écraser l'attribut href du lien Avis via le paramètre termsLink.

Vous avez également la possibilité d'écraser l'attribut href du lien Confidentialité via le paramètre privacyLink.

Ceci est configuré dans la section footer de votre page.

defFooter.outerHTML = wet.builder.footer({
...
    "hideFooterCorporate": true,
    "termsLink": {
        "href": "custom.html"
    },
    "privacyLink": {
        "href": "custom.html"
    }
...
});

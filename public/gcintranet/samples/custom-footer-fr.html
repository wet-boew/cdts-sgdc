Pied de page personnalisé

Lorsque vous utilisez le modèle d'application, vous avez la possibilité d'ajouter un pied de page personnalisé qui contiendra les liens de votre choix.

Mise en œuvre

La personnalisation du pied de page se fait via le paramètre footerSections dans la section appFooter de votre page.

  • href: l'url du lien
  • text: le texte du lien qui s'affiche
  • newWindow: Facultatif: défini sur true si vous souhaitez que le lien s'ouvre dans une nouvelle fenêtre
defFooter.outerHTML = wet.builder.appFooter({
...
    "footerSections": [{
        "href": "#",
        "text": "Lien de pied de page du portail 1",
        "newWindow": true
    }, {
        "href": "#",
        "text": "Lien de pied de page du portail 2"
    }, {
        "href": "#",
        "text": "Lien de pied de page du portail 3"
    }]
...
});

Vous avez également la possibilité d'organiser les liens en sections. Le titre des sections peut être défini via le paramètre sectionName. Lors de la création de sections, les liens sont ajoutés via le paramètre customFooterLinks.

defFooter.outerHTML = wet.builder.appFooter({
...
    "footerSections":[{
        "sectionName":"Section personnalisée 1",
        "customFooterLinks":[{
            "href":"#",
            "text":"lien1-1-nouvelle fenêtre",
            "newWindow":true
        }]
    }, {

        "sectionName":"Section personnalisée 2",
        "customFooterLinks":[{
            "href":"#",
            "text":"lien2-1-nouvelle fenêtre",
            "newWindow":true
        }]
    }, {

        "sectionName":"Section personnalisée 3",
        "customFooterLinks":[{
            "href":"#",
            "text":"lien1-1-nouvelle fenêtre",
            "newWindow":true
        }]
    }]
...
});
